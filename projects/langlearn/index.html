LANGLEARN @ EVALITA 2023

September 2023 · Alessio Miaschi

Description

This is the official webpage of the LANGLEARN - Language Learning Development Shared Task organized at EVALITA 2023. LANGLEARN was proposed at EVALITA as the first shared task on automatic language development assessment. The task consists of predicting the relative order of two essays written by the same student.

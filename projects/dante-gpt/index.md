### Description

This is the official repository for the graphical interface for text generation with the GePpeTto language model, specialized in texts in Dantean rhyme.
The interface allows the user to enter a query, to which the model responds in Dante’s writing style.
The interface and the GPT-Dante model were developed and used for the following outreach activities: 
<ul>
<li>Workshop “Computational Linguistics and Digital Philology in the Age of AI” (Bright Night, Pisa, September 29, 2023)</li>
<li>Workshop “Is it Dante or not Dante? That is the question” (Festival della Scienza, Genoa, October 27–29, 2021)</li>
</ul>
